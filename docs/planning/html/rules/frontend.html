Wonder Harness/Rules/frontend
메타 규칙 · Thymeleaf / Kendo / ES6

frontend.md

Thymeleaf·Kendo UI·ES6 프론트엔드 레이어의 컨벤션을 정의한다. JSP·jQuery 사용을 명시적으로 금지한다.

개요

ruler generate 모드가 ADR + 이 메타 규칙으로 .claude/rules/frontend.md를 생성한다.

현재 동작

  • 스택 Thymeleaf + Kendo 웹 컴포넌트 + Bootstrap 5 + ES6 모듈 (JSP·jQuery 금지)
  • 파일 구조 도메인당 {domainName}.html + {domainName}.js
  • 네이밍 JS 변수 camelCase, HTML ID kebab-case, 그리드 요소 {gridId}
  • 필수 폼 필드 CSRF hidden input, 사용자 컨텍스트 필드
  • 권한 가드 Thymeleaf sec:authorize + JS 데이터 변경 전 권한 확인
  • 저장 플로우 검증 → 확인 → POST → 토스트 알림 → 그리드 새로고침
저장 플로우: 검증→확인 다이얼로그→POST→토스트→그리드 refresh

입력 / 출력

내용
입력.claude/adr/frontend.md + 이 메타 규칙
출력.claude/rules/frontend.md

의존성

  • ruler generate 모드 + .claude/adr/frontend.md 선행 필요.
  • 프로젝트별 규칙이 이 메타 규칙보다 우선.

개선 방향

Kendo 위젯별 초기화·destroy 패턴 추가 (Grid, DropDownList, DatePicker 라이프사이클).
재사용 컴포넌트용 Thymeleaf fragment 포함 패턴 문서화.
Bootstrap 반응형 레이아웃 컨벤션 (사용 브레이크포인트·그리드 컬럼 수) 추가.
날짜·숫자 형식의 i18n/l10n 컨벤션 추가.
