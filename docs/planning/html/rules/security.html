메타 규칙 · 전 레이어 공통

security.md

전 레이어에 걸친 보안 제약을 정의한다. 권한 검증·입력 유효성·파일 업로드·민감 데이터 처리를 포함한다.

개요

ruler generate 모드가 ADR + 이 메타 규칙으로 .claude/rules/security.md를 생성한다. 특정 레이어가 아닌 모든 레이어에 적용되는 교차 관심사.

현재 동작

  • 데이터 접근 무결성 SP 전용 쓰기; 직접 SQL 구성 금지
  • 권한 검증 @PreAuthorize 전 컨트롤러 메서드; sec:authorize 뷰 섹션
  • 사용자 컨텍스트 SecurityContextHolder 경유; 요청 파라미터에서 직접 읽기 금지
  • 입력 유효성 @Valid + Form 제약 어노테이션
  • 파일 업로드 MIME 타입 검증, 크기 제한, 파일명 살균, 저장 경로 제한
  • 민감 데이터 로그에 PII 금지, 평문 비밀번호 금지, 민감 필드 마스킹
  • 에러 응답 클라이언트에는 일반 메시지; 서버 로그에만 상세 정보

입력 / 출력

내용
입력.claude/adr/security.md + 이 메타 규칙
출력.claude/rules/security.md

의존성

  • ruler generate 모드 + .claude/adr/security.md 선행 필요.
  • 프로젝트별 규칙이 이 메타 규칙보다 우선.

개선 방향

각 항목에 OWASP Top 10 카테고리 매핑 추가.
AJAX 요청의 CSRF 토큰 처리 패턴 추가 (Kendo DataSource 설정 포함).
MyBatis SP 파라미터 바인딩의 SQL 인젝션 방지 세부 사항 추가.
세션 고정(Session Fixation)·세션 타임아웃 설정 컨벤션 추가.
감사 로깅 컨벤션 (누가·무엇을·언제·어디서) 추가.
