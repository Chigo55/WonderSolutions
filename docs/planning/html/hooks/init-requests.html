init-requests

세션 시작 시 요청 시드 파일을 .claude/requests/에 자동 복사해 사용자가 매번 수동으로 준비하지 않아도 되게 한다.

개요

SessionStart 이벤트에 반응해 create_request.md와 modify_request.md 시드 파일이 없을 경우 자동 복사한다. 비차단 — 세션 시작을 방해하지 않는다.

현재 동작

  • SessionStart 이벤트 수신.
  • .claude/requests/create_request.md·modify_request.md 존재 여부 확인.
  • 없으면 plugins/wonder-harness/requests/에서 복사.
  • 비차단; 세션 시작 간섭 없음. 타임아웃 5초.

입력 / 출력

내용
입력SessionStart 이벤트
출력.claude/requests/create_request.md, modify_request.md (없을 경우에만)

의존성

  • 시드 파일이 plugins/wonder-harness/requests/에 존재해야 한다.
  • .claude/requests/ 디렉토리 쓰기 권한 필요.

개선 방향

기존 요청 파일이 30일 이상 경과한 경우 갱신 제안.
복사된 파일 vs 이미 존재하는 파일 로그 출력으로 투명성 제공.
프로젝트 로컬 버전으로 시드를 재정의하는 커스텀 템플릿 지원.
