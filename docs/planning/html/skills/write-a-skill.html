write-a-skill

새로운 SKILL.md 스킬을 작성하는 방법을 안내하는 메타 스킬. 구조·프로그레시브 디스클로저·품질 기준을 제공한다.

개요

/write-a-skill <설명>으로 호출. SKILL.md 정규 구조를 안내하고 충분한 정보가 모이면 파일을 스캐폴딩한다. 작성된 스킬은 skills/<name>/SKILL.md에 저장된다.

현재 동작

  • SKILL.md 정규 구조 제공: 프론트매터·트리거 조건·단계별 지시·예시.
  • 프로그레시브 디스클로저 패턴 설명 (최소 먼저, 필요 시 확장).
  • 유틸리티 스크립트 컨벤션 안내 (스킬이 스크립트 포함 시).
  • 위치 강제: skills/<name>/SKILL.md.
  • 충분한 정보 수집 후 SKILL.md 파일 스캐폴딩.

입력 / 출력

내용
입력스킬 설명 또는 의도 (인수)
출력skills/<name>/SKILL.md 파일

의존성

  • 플러그인 디렉토리 쓰기 권한 필요.
  • plugin-dev:skill-reviewer 에이전트로 생성 후 검토 가능 (선택).

개선 방향

생성된 SKILL.md를 품질 기준(트리거 명확성·단계 완결성·예시 포함)으로 자동 검증.
plugin-dev:skill-reviewer 에이전트와 생성 후 자동 연동.
SKILL.md 린터: 프론트매터 스키마·필수 섹션·트리거 구문 명확성 확인.
공통 패턴용 스킬 템플릿 지원 (파이프라인 스킬·리뷰 스킬·커뮤니케이션 스타일 스킬).
