handoff

현재 대화를 구조화 핸드오프 문서로 압축해 다음 세션이 전체 컨텍스트를 빠르게 복구하도록 돕는다.

개요

/handoff로 호출. 현재 대화를 구조화 마크다운으로 요약해 handoff.md에 작성한다. 다음 세션의 에이전트가 이 파일을 읽어 작업을 이어받는다.

현재 동작

핸드오프 문서 포함 내용:

  • 작업 설명
  • 내려진 결정 사항
  • 완료된 작업
  • 남은 작업
  • 관련 파일 경로
  • 다음 단계 제안
  • 다음 세션 권장 스킬

입력 / 출력

내용
입력현재 대화 컨텍스트 (인수 없음)
출력handoff.md (쓰기 권한 없으면 대화 출력)

의존성

  • 대화 컨텍스트 필수.
  • 프로젝트 루트 또는 docs/ 디렉토리 쓰기 권한.

개선 방향

최근 수정 파일의 현재 내용 스냅샷을 대화 요약과 함께 포함.
대화 내 언급된 파일 경로를 자동 탐지해 핸드오프에 포함.
--resume 커맨드: 기존 handoff.md를 읽어 세션 컨텍스트 자동 복구.
핸드오프 파일 버전 관리: handoff-YYYYMMDD-HHMMSS.md로 이력 보존.
