caveman

불필요한 표현을 제거하고 단편 문장을 허용해 토큰을 ~75% 절감하는 커뮤니케이션 압축 스킬.

개요

컨텍스트 창 보존이 중요한 긴 세션에서 유용하다. 코드 출력은 영향 없음 — 응답 텍스트만 압축된다.

일반 모드

"저는 그 파일을 읽어서 문제를 파악해 보겠습니다. 잠시만 기다려 주세요."

caveman 모드

파일 읽음. 문제 파악 중.

현재 동작

  • 사과·인사·"물론이죠" 등 필러 표현 제거.
  • 단편 문장 및 축약어 허용.
  • 코드 블록은 완전하고 정확하게 유지.
  • 세션 종료 또는 명시적 비활성화까지 지속.

입력 / 출력

내용
입력사용자 호출 (인수 없음)
출력이후 모든 응답의 스타일 변경 (신규 파일 없음)

의존성

  • 파일·상태 의존성 없음.
  • 다른 활성 스킬과 함께 작동 가능.

개선 방향

압축 레벨 설정: --light(필러만), --medium(단편 허용), --max(한 단어 가능).
/caveman off로 세션 재시작 없이 비활성화.
