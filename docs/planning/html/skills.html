Skills

특정 상호작용 패턴을 위한 재사용 가능한 SKILL.md 스킬 4개. 사용자가 명시적으로 호출하며 온디맨드로 로드된다.

개요

스킬목적상호작용 유형
caveman~75% 토큰 압축 커뮤니케이션커뮤니케이션 스타일
grill-me계획·설계에 대한 집요한 인터뷰기획 규율
handoff대화 압축 핸드오프 문서 생성컨텍스트 관리
write-a-skill새 스킬 작성 가이드메타 / 툴링

현재 동작

  • 모든 스킬은 SKILL.md 포맷이며 프로그레시브 디스클로저 구조를 따른다.
  • 스킬은 활성 세션에 로드되어 세션 종료까지 에이전트 동작을 수정한다.
  • grill-me: 한 번에 하나의 질문, 권장 답변 제시, 결정 트리 완전 해소.
  • handoff: 새 세션을 부트스트랩할 수 있는 구조화된 문서 생성.
  • caveman: 불필요한 표현 제거, 단편적 문장 허용.
  • write-a-skill: SKILL.md 구조와 품질 기준을 안내하고 파일을 스캐폴딩.

입력 / 출력

스킬입력출력
caveman사용자 호출 (인수 없음)응답 스타일 변경 (파일 없음)
grill-me계획·설계 설명 (인수)Q&A 세션 → 공유 이해
handoff현재 대화 컨텍스트handoff.md 문서
write-a-skill스킬 설명 (인수)skills/<name>/SKILL.md 파일

의존성

  • 스킬은 독립적 — 스킬 간 상호 의존성 없음.
  • grill-me·handoff는 대화 컨텍스트에 의존.
  • write-a-skill은 플러그인 디렉토리 쓰기 권한 필요.

개선 방향

grill-me가 해소된 Q&A를 자동으로 기획 문서로 저장하는 기능 추가.
handoff에 최근 수정 파일의 현재 내용 스냅샷 포함.
caveman에 압축 레벨 설정 기능 (--light / --medium / --max).
기존 스킬을 품질 기준으로 평가하는 review-skill 스킬 추가.
